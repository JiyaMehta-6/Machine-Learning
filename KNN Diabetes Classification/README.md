# 🧠 KNN Diabetes Classification

A machine learning project that uses the k-Nearest Neighbors (KNN) algorithm to predict whether a patient has diabetes based on medical diagnostic measurements. This repo exists to evaluate how distance-based classification performs on a real-world health dataset.

## 🧠 Project Overview

  - Goal: Predict whether a patient has diabetes based on clinical features.
  
  - Approach: Apply KNN classification with proper preprocessing and parameter tuning.
  
  - Outcome: Evaluate model performance using standard classification metrics and visualizations.

## 🚀 Features

- Data preprocessing and cleaning

- Feature scaling (essential for KNN performance)

- KNN model training with different values of k

- Model evaluation using classification metrics

- Visualization of performance results

## 📊 Model Used

- k-Nearest Neighbors (KNN)

- Model performance is measured using:

- Accuracy

- Precision

- Recall

- F1-score

- Confusion Matrix

## 📈 Results

Visualizations and performance summaries are stored in the visuals/ directory. These illustrate how different k values and preprocessing steps affect classification performance.
