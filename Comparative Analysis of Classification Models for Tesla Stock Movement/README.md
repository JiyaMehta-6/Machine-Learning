# 📊 Comparative Analysis of Classification Models for Tesla Stock Movement

A machine learning project that compares different classification models to predict Tesla (TSLA) stock price movement based on historical data and engineered features. This repo exists so you can judge whether your model actually learned something useful or just memorized noise.

## 🧠 Project Overview

  - Goal: Predict whether Tesla stock price goes up or down on a given day.

  - Approach: Train and evaluate multiple classification models.

  - Outcome: Performance comparison across models using metrics and visualization.

## 🚀 Features

  - Data collection from historical price sources
  
  - Feature engineering (momentum, return, moving averages, volatility)
  
  - Multiple classification models trained
  
  - Model evaluation with metrics & visual comparison

## 📊 Models Compared

  - Logistic Regression
  
  - Random Forest Classifier
  
  - Support Vector Machine
  
  - k-Nearest Neighbors


Each model’s performance is measured by accuracy, F1-score, and confusion matrices.

## 📈 Results

Visualizations and performance summaries are stored in the visuals/ directory. Look there to see which model (if any) actually beats guessing.

### This is an academic experiment, not financial advice. Always validate models on fresh data before any real-world use (please).
